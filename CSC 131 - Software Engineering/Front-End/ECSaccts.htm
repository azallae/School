ECS Computing Network Accounts

Why do I need an ECS Account?

  • To work on class assignments
  • Free web-based E-Mail (Zimbra)
  • 1.5GB of personal storage space
  • Access to all ECS Workstations and Servers including Windows, Unix, HPUX and WebOS
  • Free webspace to build websites or applications using Open Source programming tools such as PHP, MySQL
  • Access ECS licensed software including downloads from Microsoft's MSDN Alliance
  • It is active until six months after graduation

Request or Modify

  • Apply for a new ECS account and/or a MySQL Database
  • Request new password for your account
  • Change your account's disk quota
